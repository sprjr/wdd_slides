About WDD_Slides

WDD_Slides is a project that has a few goals involved with its creation and existence. I wanted to create a lightweight application that was fully based in a cloud environment (e.g. everything created or developed from within a browser / web application).

To do that I utilized (currently) Cloud9IDE, CouchDB (hosted by Cloudant) and GitHub. This allowed me to develop my app simply, and still retain version control / easy open source sharing, as well as provide me with a dynamic content database for the project itself.

The Second Part

The other part of this project was intended to pass on to future students in my degree program at Full Sail University so that they could learn from my own failures and perhaps successes in the program. The degree itself (like any college) isn't cheap, and I wanted to be sure to maximize what I took away from it as well as what I could give back so others can benefit as well.

I currently plan to implement a category / author sorting feature which should actually be quite easy to do, as well as an admin control panel to input new slide content into the Couch (however that is proving difficult to do with my current tech lockdown). Perhaps if a solution doesn't present itself I will break at least the "cloud only" rule and whip up a quick admin panel and shoot it off to Cloudant with a replicate and be done.
